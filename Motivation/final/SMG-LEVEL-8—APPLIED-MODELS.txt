4. FEASIBLE ACTION SET

Define survival-feasible correspondence:

Ω_S(X_t) := { a ∈ Ω(S_t) | R(S_t, a) ≤ τ_S(t) }

Assume:
- Ω(S_t) is non-empty and compact
- R(S_t, a) is lower semicontinuous in a

⇒ Ω_S(X_t) is non-empty and compact (when feasible)

Fallback rule:
If Ω_S(X_t) = ∅:
    a_t := argmin_{a ∈ Ω(S_t)} R(S_t, a)

5. VALUE CONSTRUCTION

5.1 Projection Operator

Proj_{M_t}(v) ∈ argmin_{u ∈ M_t} ||u − v||

Assume:
- V is a Hilbert space OR M_t is closed and convex
⇒ projection exists
⇒ measurable selector exists

--------------------------------------------------

5.2 Meaning-Projected Value

V_M(a) := E_{s ~ P_t}[ Proj_{M_t}(V(s,a)) ]

--------------------------------------------------

5.3 Objective Function

J_t(a) := V_M(a)
          − C(a)
          − λ_S(t) · R(S_t,a)
          − λ_M(t) · D_M(V_M(a))

--------------------------------------------------

5.4 Exploration-Augmented Objective (Optional but Recommended)

J'_t(a) := J_t(a) + β_t · H(P_{t+1}(· | S_t,a))

Where:
- H(P) = −Σ P log P
- β_t ≥ 0
- β_t → 0, Σ β_t = ∞, Σ β_t² < ∞

6. VALUE FUNCTION AND POLICY
