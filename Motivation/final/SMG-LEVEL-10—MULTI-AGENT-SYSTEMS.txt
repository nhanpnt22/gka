SURVIVAL–MEANING–GAIN (SMG) SYSTEM
LEVEL 10 — MULTI-AGENT SYSTEMS

Author: Phạm Ngọc Thanh Nhân (Brian Pham)
Date: March 24, 2026
--------------------------------------------------

1. ONTOLOGICAL SETTING

Let:

- I = {1, ..., N} be a finite set of agents
- S be the global state space (assumed standard Borel)
- A_i be the action space of agent i (standard Borel)
- A = ×_{i ∈ I} A_i be the joint action space

Dynamics:

T : S × A → 𝒫(S)

is a measurable stochastic kernel.

Observations:

For each agent i:

O_i : S → 𝒫(𝒪_i)

is a measurable observation mapping.

--------------------------------------------------

2. INFORMATION STRUCTURE

Each agent i has an information space:

ℐ_i ⊆ ⋃_{t≥0} (𝒪_i × A_i)^t × 𝒪_i

ℐ_i ≠ ∅

Information evolves based only on:

- private observations
- own past actions

No global information access is assumed.

--------------------------------------------------

3. BELIEF HIERARCHIES

Each agent i maintains beliefs over:

- global states
- other agents’ types

Define type space:

Θ_j = (π_j, K_j, M_j)

where:
- π_j : ℐ_j → 𝒫(A_j) is a measurable policy
- K_j is an epistemic configuration (Level 2)
- M_j ⊆ V_j is identity

Belief mapping:

b_i : ℐ_i → 𝒫(S × Θ_{−i})

Beliefs extend to infinite hierarchies:

Θ_{−i} includes recursive beliefs over types.

Assumption:

The hierarchy is constructed over standard Borel spaces,
ensuring existence via projective limits or equivalent type-space constructions.

--------------------------------------------------

4. POLICY STRUCTURE

Each agent i follows a policy:

π_i : ℐ_i → 𝒫(A_i)

Policies are:

- measurable w.r.t ℐ_i
- adapted to available information

No optimality is assumed.

--------------------------------------------------

5. VALUE (SMG-CONSISTENT)

Let τ denote a trajectory induced by:

(S_0, {π_i}_{i ∈ I}, T)

Each agent evaluates via:

v_i := G_meta^i(τ, {M_{i,t}}_{t≥0})

where:

- G_meta^i operates over full trajectories
- explicitly depends on identity evolution
- incorporates:
  - survival constraints
  - meaning (identity alignment)
  - gain

No reduction to additive reward or expectation is imposed.

--------------------------------------------------

6. IDENTITY DYNAMICS

Each agent has identity:

M_{i,t} ⊆ V_i

Evolution:

M_{i,t+1} = Φ_i(M_{i,t}, ℐ_{i,t}, b_{i,t})

Constraint:

d_{M_i}(M_{i,t}, M_{i,t+1}) ≤ δ_i

Identity evolution is:

- endogenous  
- influenced by beliefs and interaction  
- unconstrained by external enforcement  

--------------------------------------------------

7. INTERACTION DYNAMICS

A trajectory τ includes:

- state trajectory {S_t}
- information trajectories {ℐ_i,t}
- belief trajectories {b_i,t}
- identity trajectories {M_i,t}

Generated by:

- initial state S_0
- policies {π_i}
- transition kernel T
- observation mappings {O_i}

--------------------------------------------------

8. ADMISSIBLE DEVIATIONS

A deviation policy π_i′ is admissible if:

- π_i′ : ℐ_i → 𝒫(A_i) is measurable
- π_i′ respects the information structure ℐ_i
- induced identity evolution via Φ_i remains well-defined
- resulting trajectory distributions are measurable

Only admissible deviations are considered in equilibrium.

--------------------------------------------------

9. EQUILIBRIUM (TRAJECTORY-BASED)

A policy profile {π_i} is an equilibrium if:

For all agents i and all admissible deviations π_i′:

Let τ be induced by {π_i}  
Let τ′ be induced by (π_i′, π_{−i})

Then:

G_meta^i(τ, {M_{i,t}}) ≥ G_meta^i(τ′, {M'_{i,t}})

where evaluation includes:

- belief evolution  
- identity evolution  
- downstream trajectory effects  

Equilibrium is defined over trajectory distributions.

--------------------------------------------------

10. COALITIONS

A coalition C ⊆ I is a coordinating subset.

Coalition policy:

π_C : ×_{i ∈ C} ℐ_i → 𝒫(×_{i ∈ C} A_i)

Evaluation:

Each i ∈ C uses G_meta^i.

Internal Stability:

C is internally stable if:

For all i ∈ C and admissible π_i′:

G_meta^i(τ_C) ≥ G_meta^i(τ_{C,i′})

This defines resistance to unilateral deviation within C.

Remark:

External stability (e.g., blocking coalitions or core concepts)
is not imposed at this level.

--------------------------------------------------

11. EMERGENCE

Macro-structures arise endogenously:

- norms: stable policy regularities  
- institutions: persistent coalitions  
- conventions: aligned belief-policy patterns  

These emerge from:

- repeated interaction  
- belief dynamics  
- identity evolution  

No external imposition occurs.

--------------------------------------------------

12. FEASIBILITY AND STRUCTURAL ASSUMPTIONS

Assume:

- S, A_i, Θ_i are standard Borel spaces  
- T is a measurable stochastic kernel  
- O_i are measurable  
- π_i are measurable  
- b_i are measurable  

No additional constraints are imposed on:

- communication  
- coordination capacity  
- policy complexity  

These are intentionally left unrestricted.

--------------------------------------------------

13. SYSTEM DEFINITION

The Level 10 SMG system is:

𝒮_MA =
(I, S, {A_i}, T, {O_i}, {ℐ_i}, {b_i}, {π_i}, {G_meta^i}, {M_i})

This defines:

- agents  
- environment  
- information  
- beliefs  
- policies  
- identity dynamics  
- value structure  

--------------------------------------------------

14. INTERPRETATION

At Level 10:

- Agents operate under uncertainty (Level 2)  
- Evaluate via SMG meta-objectives (Level 3+)  
- Evolve identity dynamically (Level 9)  
- Interact strategically with others  

Outcomes are:

- trajectory-dependent  
- belief-coupled  
- identity-coupled  

Equilibrium is a property of trajectories,
not static action profiles.
